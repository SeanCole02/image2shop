# Mockups

Create one folder per page slug:

- `<page-slug>/desktop/`
- `<page-slug>/mobile/`
- `<page-slug>/notes.md`

Use page slugs that are lowercase and hyphenated, such as `homepage` or `product-detail`.

Put desktop references in `desktop/` and mobile references in `mobile/` so the related page spec can point to them consistently.
