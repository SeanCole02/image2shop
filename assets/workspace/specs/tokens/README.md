# Tokens

Put shared design tokens here.

Use `theme-tokens.json` for:

- brand colors and supporting colors
- extra brand-specific color roles under `colors_additional`
- typography choices
- extra brand-specific type roles under `typography_additional`
- spacing and radius values
- shared layout values
- motion values when needed

Keep these values here because they should stay consistent across the whole theme, not drift page by page.
