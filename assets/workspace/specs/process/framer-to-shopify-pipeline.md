# Framer to Shopify Pipeline

Goal:

- convert mockups into a maintainable Shopify Liquid theme with clear merchant-editable controls

Inputs:

- Framer link or source reference
- desktop and mobile mockups
- copy and image assets
- app requirements

Phases:

1. shared theme planning
2. intake
3. component mapping
4. token extraction
5. theme architecture
6. implementation
7. QA
